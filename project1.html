About Project

Project Greensnake adalah nama temporer dari game pixel art 2 dimensi bergenre puzzle adventurer untuk platform Windows dan Android yang sedang kukembangkan menggunakan game engine Unity. Pengembangan video game ini terinspirasi dari game petualangan puzzle retro yang pernah populer pada masanya, Tentunya ditambah dengan sentuhan modern dan penuh warna! Game ini juga kukembangkan dengan tujuan untuk mempelajari dunia game development dan tentunya juga sebagai salah satu proyek portofolio pribadiku.

Tapi hingga saat ini, Belum terdapat versi release / full karena masih dalam tahap pengambangan awal. Tetapi sudah tersedia versi uji coba untuk platform Android!
